AI Email Writer

An AI-powered Email Writer application that helps users generate professional and personalized emails instantly using Generative AI. This project consists of a Spring Boot backend, a React frontend, and a browser extension.

🚀 Features
Generate professional emails using AI
User-friendly React frontend
Spring Boot REST API backend
Browser extension integration
Fast and responsive UI
Clean and modular project structure

⚙️ Installation & Setup
1️⃣ Clone Repository
git clone https://github.com/MADHURI-HS/email-assistance.git
cd email-assistance
▶️ Run Backend
cd email-writer-sb

Run Spring Boot application:

./mvnw spring-boot:run

Backend runs on:

http://localhost:8080
▶️ Run Frontend
cd email-write-react

Install dependencies:

npm install

Run application:

npm run dev

Frontend runs on:

http://localhost:5173
